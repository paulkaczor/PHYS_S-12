Assignment 4: Rapid Prototyping

Create a kinetic sculpture using an electronic circuit with physical components.

Planning the Design

I was inspired by the rotating abstract sculptures shown as examples in class. However, I wanted to try to create two axes, that interlocked as they rotated. Also, I wanted to try to create my sculpture with ellipse pieces, rather than striaght lines. The first step to creating my kinetic sculpture was to develop my design. I decided the four general patterns I wanted to be shown as the two axes rotated together. Once I decided on the pattern, I decided on exact measuements for each of the four radii of my ellipse.

Making the Pieces

To make the elliptical pieces, I first measured the width of the cardboard I was going to use. This gave me a rough estimate for the size of the entire sculpture. Next, to create my pieces, I used the compass to create in inner and outer radius for the piece. Dividing the piece into quarters, one quarter was designated as the long side and the oppsoite was for the short side. Next, I made a stencil to connect the to quarters. I completed this for each of the different sized pieces. Finally, I cut out the pieces using an exacto knife, drilled a hole at the center, and taped all the way around the cardboard.

Setting Up the Sculpture

Now with the pieces created, I moved on to the rest of the sculpture. First, I spaced out the pieces on the wooden dowells so that they would interlock with eachother as they spun. Now knowing the size, I cut out the box and drilled holes 9 cm apart from eachother. Taping the box together, with the dowells inside, the pieces could rotate freely. Next, I had to cut off the excess length of the dowells, and attach the motors. To attach the motors to the dowells, I drilled a hole through the dowell and through the 3D printed motor attachment. Putting a nail through both the motors when powered on would rotate the dowells. However, when trying to rotate both axes at the same time, there were prblems. Becasue the cardboard was not perfectly level, and the tape added texture and lots of width in some spaces, the two axes constatnly caught on each other. To combat this, I had to remove an ellipse on each axis and redistribute all the ellipses. Now with the axes truly free to rotate, I moved onto the circuitry to power the motors.

Circuitry

To create the circuitry to power the motors, I started with the basic functioned motor code as discussed in class. I had to add outputs and a second function for the second motor. This a simple copy and paste with a few edits. As the full output of the motor seemed a little too strong at first, I also changed the code to analogWrite to tone down the power output. This code worked, and is linked here: Kinetic Sculpture Code. However, I had more success with the potentiometer controlled circuit. By following the directions on the class website and ajusting the code slightly for a second motor, I was able use the potentiometer to control the speed of my sculpture's rotation. The code for the potentiometer controlled rotation is here: Potentiometer Controlled Sculpture. Using this potentiometer, I was able to turn the motors off quuickly if the pieces on the different axes collided. This was very important, especially for early in my sculpture design, when I was still adjusting the spacing between the pieces. Using the multimeter, I was able to measure the voltages with respect to ground at all different points along the circuit. Becasue I was using a potentiometer, there was a range of voltages at different points (full resistance - no resistance). Interstingly, decreasing the potentiometer's resistance did not exclusively increase the voltage with respect to ground at all points. I was also not able to exactly measure the current across the potentiometer, becasue I was unsure of it's exact resistance. Therefore, I replaced the potentiometer with a 100 Ohm resitor. Now, measuring the voltage drop across the resistor and using I=V/R, I found the current in this circuit to be 8.2x10-3 Amps.

See Assignment 3 to see how I used the Microcontroller Analog in (ADC) to measure the voltage of a potentiometer and voltage-divider.

Ultimately, the sculpture was able to rotate using the motors. However, the motors were not powerful enough to turn the axes as well as I wanted. As shown in the video, the motors make the sculpture rotate slowly and choppy. In fact, manually rotating the axes ended up with the best result.
